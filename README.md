# 本アプリについて

本アプリは **競馬予想投稿アプリ** です。
レースを選んで馬印の作成、買い目、予想コメントをみんなにシェアしよう！
また、気になるユーザをフォロー、予想をお気に入り登録して実際に馬券を購入する際に参考にすることもできます。
